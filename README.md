# Github Repositories Timeline

This app generates a timeline with an entry for each repository the selected user has created.

## Stack

-  React
-  React Router
-  Redux
-  redux-thunk
-  Axios
-  Semantic UI
-  Github APIv3:
   -  https://api.github.com/users/:user
   -  https://api.github.com/users/:user/repos
